The TV_annotation.xlsx file provides metadata and annotation information for each audio recording included in the Variatica dataset. This file serves as the central reference for accessing and understanding the content of the dataset.

ğŸ“‹ Contents

Each row in the Excel file corresponds to a single audio recording. The following columns are included:

Composer: Name of the composer of the piece.

Period: The historical period or style (e.g., Classical, Romantic).

Link: A direct link to the publicly available YouTube video of the performance.

Boundary: Detailed structural boundaries of the theme and variation form.

Label: The label of the theme and variation form's structural segments, such as Theme (T), 1st Variation (V), Coda (C).

Applause begin/end: Indicates whether and where audience applause occurs in the audio.

Instruments: The instrument(s) used in the performance (e.g., piano, string quartet, orchestra).

This file can be used to:

Access original performances for audio-based analysis.

Align structural annotations with audio for training or evaluation.

Filter or analyse pieces by composer, period, or instrumentation for comparative studies.

Align the annotations of the applause to the audio when needed.
